## MateNN this is my open source ai bot. You can ask it questions, chat with it and other similar thinks... For now, it's in the development stage, but i will improve it over time.
_____
***Guide for installing:***

```git clone https://github.com/seeyoualways/MateNN.git MateNN```

```cd MateNN/```

1. System update and installation of required packages

for a Arch linux:

```sudo chmod +x arch.sh```

```./arch.sh```


for a Debian/Ubuntu linux:

```sudo chmod +x debu.sh```

```./debu.sh```


2. Program launch

```python MateNN.py```
